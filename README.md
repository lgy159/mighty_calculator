# Mighy Counter

## :loudspeaker: Description
마이티(보드게임)의 오프라인 플레이 시 번거롭던 점수 계산을 앱으로 할 수 있게 도와줍니다.
현재 PlayStore 검토 중이며 웹 도메인 또한 배포 예정입니다.

## :mega: 주요 기능
  - 마이티 점수 계산
  - History 확인 가능
  - 플레이어 이름 변경 가능
